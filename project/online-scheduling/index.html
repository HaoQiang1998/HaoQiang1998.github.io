Online Scheduling Algorithm Design in Edge Computing System

with Liuyan Liu, Chi Zhang, Wanli Cao, Haisheng Tan, etc.
Jan 1, 0001

In this project, we aim at working out some practical online scheduling algorithms to obtain effective task scheduling in edge computing system. Modern applications are complicate and always latency-sensitive. Edge computing is a promising computing paradigm aiming at reducing communication latency within the application and supporting applications of Internet of Things (IoT). This project is motivated by the need of scheduling strategy specific to Edge Computing and already had two papers1 published.
